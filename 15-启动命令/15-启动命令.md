1. 启动 systemctl start docker
2. 停止 systemctl stop docker
3. 重启 systemctl restart docker
4. 查看状态 systemctl status docker
5. 查看概要信息 docker info
6. 查看总体帮助文档 docker --help
7. 查看命令帮助文档 docker <command> --help
